# StickyMVP 수동 QA 체크리스트 (스티커 전용 UI)

## 사전 준비
- macOS 13 이상
- 앱 최초 실행 시 접근성/입력 모니터링 권한 요청이 뜨면 허용

## 기능 점검
1. 앱 실행
- 앱 실행 후 사이드바 없이 스티커 캔버스만 보이는지 확인
- 기본 스티커가 1개 이상 보이는지 확인

2. 새 스티커 생성
- 좌상단 `새 스티커` 버튼 클릭 시 새 스티커 생성 확인
- 메뉴바 `새 메모 만들기` 클릭 시 새 스티커 생성 확인
- 단축키 `Cmd + Option + N` 입력 시 새 스티커 생성 확인

3. 스티커 편집
- 본문 텍스트 입력 시 즉시 반영 확인
- 글자 크기 슬라이더 조절 시 텍스트 크기 즉시 변경 확인
- 글자 색 원 클릭 시 텍스트 색상 변경 확인

4. 스티커 스타일/크기
- 스티커 색 원 클릭 시 배경색 즉시 변경 확인
- 우하단 리사이즈 아이콘 드래그로 스티커 크기 변경 확인
- 스티커 카드 드래그로 위치 이동 확인

5. 삭제
- 스티커 우상단 `x` 버튼 클릭 시 즉시 삭제 확인

6. 항상 위 고정
- 메뉴바 `항상 위에 고정` 토글 ON/OFF 시 창 레벨 변경 확인
- ON 상태에서 다른 앱 창 위에 StickyMVP가 유지되는지 확인

7. 영속성
- 앱 종료 후 재실행
- 텍스트/글자크기/글자색/스티커색/위치/크기가 유지되는지 확인

## 합격 기준
- 위 1~7 항목이 모두 통과하면 현재 요구사항 충족

# MustDo 빠른 시작 가이드

## 1) 설치
1. `dist/release/MustDo-0.1.0.dmg`를 열기
2. `MustDo.app`을 `Applications`로 드래그
3. 앱 실행

## 2) 기본 사용
1. 스티커 상단 `+` 버튼: 새 스티커 생성
2. 상단 `제목` 입력
3. 자유 메모 영역에 내용 작성
4. 체크리스트 영역 `+`로 항목 추가
5. 체크박스 선택 시 취소선 표시
6. 항목 `-` 버튼으로 체크리스트 삭제

## 3) 스티커 닫기/복원
1. 스티커 우측 상단 `x` 클릭: 해당 스티커만 닫힘
2. 파일은 자동 저장됨
3. 스티커 우측 상단 `파일 불러오기` 버튼으로 `.mustdo` 파일 열기

## 4) 저장 위치
- `~/Library/Application Support/MustDo/notes`

## 5) 자주 쓰는 옵션
1. 상단 `...` 버튼: 옵션 패널 열기
2. 글자 크기 슬라이더로 크기 조절
3. 스티커 색상 선택

## 6) 문제 해결
1. 다른 앱 위에 계속 보이면: 앱 메뉴에서 `항상 위에 고정` 해제
2. 실행 시 아무것도 안 보이면: `파일 불러오기`로 기존 `.mustdo` 복원
3. 설치 경고가 나오면: 시스템 설정 > 개인정보 보호 및 보안 > `그래도 열기`

# MustDo 기능 설명서

## 제품 개요
MustDo는 macOS에서 스티커를 개별 윈도우로 띄워 사용하는 포스트잇 앱입니다.

## 핵심 기능
1. 개별 스티커 윈도우
- 각 스티커가 독립 윈도우로 생성됨
- 스티커별 이동/크기 조절 가능
- 다른 앱 포커스 시 스티커는 뒤로 감

2. 스티커 작성/관리
- 상단 제목 입력
- 자유 메모 입력(TextEditor)
- `+`로 새 스티커 생성
- `x`로 해당 스티커만 닫기
- 닫힌 스티커는 파일에서 다시 열기 가능

3. 체크리스트
- 체크리스트 영역 `+`로 항목 생성
- 체크 시 텍스트 취소선 표시
- `-` 버튼으로 해당 체크 항목 삭제

4. 옵션
- 옵션 버튼으로 설정 영역 열기/닫기
- 글자 크기 슬라이더(기본 12)
- 스티커 색상 변경

5. 저장/복원
- 변경 시 즉시 저장 + 주기 저장(15초)
- 스티커별 파일 저장(.mustdo)
- 앱 실행 시 최근 스티커 자동 복원
- 내용 없는 기본 신규 스티커는 파일 저장 제외

## 파일 위치
- 상태 파일: `~/Library/Application Support/MustDo/notes.json`
- 스티커 파일: `~/Library/Application Support/MustDo/notes/*.mustdo`

## 릴리즈 산출물
